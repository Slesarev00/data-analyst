# Анализ корреляционных связей свойств почв (хребет Басеги) и пород (Ергаченский карьер)

**Автор:** Слесарев Николай Владимирович  
**Направление:** Анализ данных / Почвоведение

---
## 📊 Данные

| Файл | Описание | Размерность |
|------|----------|-------------|
| `soil_correlation.csv` | Корреляционная матрица свойств почв | 7×7 |
| `rock_correlation.csv` | Корреляционная матрица свойств пород | 8×8 |

---

## 📈 Методы анализа

| Метод | Применение |
|-------|------------|
| Парная линейная регрессия | Оценка зависимости между парами переменных |
| Корреляционный анализ | Построение матриц корреляции (Пирсон) |
| Иерархический кластерный анализ | Классификация горизонтов почв (метод Уорда) |
| **Метод k-средних** | Разбиение объектов на 5 кластеров по свойствам |
| Дисперсионный анализ (ANOVA) | Проверка значимости различий между кластерами |

## 🧪 Краткие результаты

### Почвы (хребет Басеги)
- Сильнейшая связь: **EKO ↔ S** (r = 0.97)
- Гумус тесно связан с физ. глиной (r = 0.92)
- Zr не зависит от высоты местности

### Породы (Ергаченский карьер)
- Сильнейшая связь: **Hг ↔ V** (r = -0.99)
- Гипс тесно связан с физ. глиной (r = 0.93)
